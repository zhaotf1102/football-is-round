# 更新日志

## v1.3 - 截图录入与简洁交互版

发布日期：2026-06-22

### 新增

- 新增截图上传入口，支持多张彩票截图或票面照片。
- 新增 OCR（Optical Character Recognition，文字识别）入口，尝试从截图中提取票面文字。
- 新增待核票草稿区，识别结果必须人工确认后才可进入模型。
- 新增快速粘贴模式，方便把聊天识别结果直接导入。
- 交互风格改为更接近体彩页面的红白简洁界面。
- 首页加入更清晰的流程入口：截图录入、EV候选、模型日志。

### 保留

- 数据门禁。
- EV（Expected Value，数学期望 / 赔率价值）计算。
- 动态仓位模块。
- 票仓生成与组合EV计算。
- 模型日志体系。

### 限制

- OCR（文字识别）依赖 CDN（Content Delivery Network，内容分发网络）加载脚本，可能受网络影响。
- 识别结果只用于减少输入量，不能替代人工核票。
- 当前仍是纯前端静态网页，未接入后端爬虫和自动数据清洗。

### 下一步建议

- v1.4：加入更强的票面结构解析。
- v1.5：加入赛果录入与自动复盘报表。
- v2.0：加入后端数据抓取、清洗、来源评分与版本化数据库。
